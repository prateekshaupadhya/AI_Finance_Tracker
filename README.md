# 💰 AI Powered Personal Finance Advisor with Behavioral Analytics

## 📌 Overview

The AI Powered Personal Finance Advisor is a Machine Learning-based application that helps users analyze their financial health and predict whether they are likely to achieve their savings goals. The system evaluates financial information such as monthly income, expenses, credit score, loan payments, investments, and savings to provide personalized financial advice.

---

## 🎯 Project Objectives

* Analyze users' financial data.
* Predict whether the savings goal will be achieved.
* Provide personalized financial recommendations.
* Visualize financial information using charts.
* Help users make better financial decisions.

---

## 🚀 Features

* 📊 Financial data analysis
* 🤖 Machine Learning prediction using Random Forest Classifier
* 💰 Savings goal prediction
* 📈 Financial summary dashboard
* 📉 Bar chart visualization
* 💡 Personalized financial advice
* 🌐 Streamlit web application

## 📂 Dataset

* **Dataset Name:** Personal Finance Tracker Dataset
* **Source:** Kaggle
* **Format:** CSV
* **Columns:** 25 financial attributes including monthly income, monthly expenses, credit score, investment amount, actual savings, and savings goal status.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/AI_Finance_Tracker.git
```

### 2. Open the project folder

```bash
cd AI_Finance_Tracker
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

### Train the model

```bash
python train_model.py
```

### Launch the Streamlit application

```bash
streamlit run app.py
```
